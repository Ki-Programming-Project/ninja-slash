# NINJA SLASH - 忍斬

A stylish, minimalist Roguelike action game built with pure vanilla JavaScript and HTML5 Canvas.

## 🔗 Live Demo
Play the game directly in your browser:  
[https://men0tai0ko.github.io/ninja-slash/]

## ⚡ Features
- **Roguelike Mechanics**: Face randomized challenges and test your skills.
- **Fluid Combat**: Dynamic action powered by a custom State Machine and Object Pooling for high performance.
- **Immersive Audio**: Dynamic sound feedback using the Web Audio API.
- **Cross-Platform**: Fully responsive design optimized for both desktop and mobile devices.
- **No Dependencies**: Built with 100% vanilla JavaScript, HTML5, and CSS3.

## 🛠 Technology Stack
- **Language**: JavaScript (ES6+)
- **Graphics**: HTML5 Canvas API
- **Audio**: Web Audio API
- **Layout**: CSS3 Flexbox / Grid (Responsive)

## 🚀 Quick Start
No build process is required. Simply clone the repository and open `index.html` in any modern web browser.
